HIGHLIGHTS OF MY WORK

I have had the opportunity to wear many hats in my career: developer, product manager, team lead, and project manager. I have enjoyed them all, but I'm looking for a technical PM or software engineering position.

  • I'm wrapping up work on JamHive which is a project that allows bands to collaborate via their web browser. My first task was to migrate the Ruby on Rails app from generic HTML to Bootstrap. Then I refactored the error-prone JavaScript enabling users to successfully record audio and upload it to our Amazon S3 store.
  • My most recent pet project was SplitDat, a bill-splitting app for friends and coworkers. I put a lot of effort into iterative improvements to the UX. We started by using GWT, but I grew wary of the hoops I had to jump through to use external JavaScript libraries. So we migrated to the hot client-side framework of the day, Backbone.js.
  • At CampusNow we tried to create interactive textbooks. For our first textbook, we created an Interactive momentum demonstration for physics students. I focused on getting the physics code working correctly as well as polishing up the UX before we released it to students.
